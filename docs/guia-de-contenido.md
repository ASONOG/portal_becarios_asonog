# Guía de Actualización de Contenido

## Contenido estático (requiere desarrollador)
Todas las páginas públicas del sitio tienen el contenido 
hardcodeado en archivos blade. Para modificar textos, 
imágenes o secciones de estas páginas es necesario 
editar directamente los archivos del proyecto y hacer 
un nuevo deploy.

Páginas con contenido estático:
- Inicio
- Programas
- Nosotros
- Contacto
- (agregar las que correspondan)

## Contenido dinámico (desde el panel de administrador)

### Galería de imágenes
La única sección actualizable sin tocar código es la 
galería de imágenes.

**Cómo agregar o cambiar imágenes:**
1. Iniciar sesión como administrador
2. Ir a la sección de Galería en el panel
3. Subir nuevas imágenes o eliminar las existentes

No se requiere conocimiento técnico para esta operación.
