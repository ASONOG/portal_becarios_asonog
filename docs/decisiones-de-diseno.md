# Decisiones de Diseño

## Dirección visual
El sitio busca un balance entre modernidad e 
institucionalidad. Se priorizó una identidad visual 
limpia y profesional que transmita confianza, 
coherente con la naturaleza de ASONOG como organización 
sin fines de lucro con trayectoria.

## Paleta de colores
Verde como color principal — asociado a crecimiento, 
esperanza y naturaleza, alineado con los valores del 
programa de becas.

## Decisiones de UX
- Animaciones AOS desactivadas en móvil para evitar 
  confusión en usuarios que ven contenido parcial
- Texto alineado a la izquierda en móvil para mejorar 
  legibilidad
- Un CTA principal por página para no dispersar 
  la atención del usuario
- Imágenes y testimonios reales de becarios para 
  generar credibilidad

## Enfoque mobile-first
El sitio fue auditado y ajustado específicamente para 
móvil dado que el público objetivo (jóvenes de 18-35 
años) accede principalmente desde teléfono.
